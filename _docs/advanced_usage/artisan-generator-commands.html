Tip:

You can use --help with any of the following commands to find its arguments and options.

Note: All of the following commands use "Blog" as an example module name, and example class/file names.

Note: Many of the generator commands also support nested folder structures. For example, if you wanted to generate a model inside a directory called "PostModel", you could call php artisan module:make-model PostModel/Post Blog. This will generate the model with the correct namespace and place it inside of Modules/Blog/Models/PostModel/Post.php.
